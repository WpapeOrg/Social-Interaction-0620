# 大版本拆分与上线计划

本文档用于将 `master` 现有能力按大版本拆分，支持后续分批上线与回归验证。

## 1. 已实现能力（已创建分支）

### V1.0 基础 MVP（可首批上线）
- 分支：`devlop-mvp-core-v1.0`
- 基线提交：`23b79a5`
- 包含功能：
  - 小程序核心页面：登录、资料、推荐、会话、聊天、我的。
  - 基础 API：登录、资料、推荐匹配、会话消息、举报拉黑。
  - 管理后台基础能力：举报处理、用户封禁/解封。
  - UI 预览稿与基础体验改造。

### V1.1 实时聊天增强
- 分支：`devlop-realtime-im-v1.1`
- 基线提交：`1a7bc54`
- 包含功能：
  - WebSocket 实时消息、已读回执。
  - 输入中状态 + 断线重连策略。
  - 离线补偿拉取、送达回执、多设备同步。

### V1.2 推送闭环能力
- 分支：`devlop-push-closedloop-v1.2`
- 基线提交：`3d79da0`
- 包含功能：
  - 通知偏好设置与离线任务队列入库。
  - 通知 Worker、指数退避重试、死信状态。
  - 微信订阅消息投递、回调签名校验、AES 解密、幂等入库、回写任务状态。

### V1.3 版本与镜像稳定性
- 分支：`devlop-version-stability-v1.3`
- 基线提交：`c1ba687`
- 包含功能：
  - Node/npm 推荐版本收敛（Node 20.19.x + npm 10.9.2）。
  - `.nvmrc`、`.npmrc`、`engines` 约束。
  - `package-lock` 镜像源与依赖拉取稳定性治理。
